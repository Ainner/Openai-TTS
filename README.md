# OpenAI TTS

## 简述

该存储库提供了一个 Gradio 界面，旨在利用 OpenAI 的文本转语音 (TTS) API。该界面允许用户通过不同的模型和语音选项，根据提供的文本生成语音。

1. 先将代码克隆下来
    ```bash
    git clone https://github.com/Ainner/Openai-TTS
    cd openai-tts
    ```

2. 安装依赖
    ```bash
    pip3 install -r requirements.txt
    ```

## Setup

1. 在 app.py 中插入 API key

2. 启动界面
    ```bash
    python3 app.py
    ```
